# FindRefs
An experimental extension for MediaWiki under experimental stage by an experimental mediawiki contributor

#Installation Instructions
*Navigate to the extensions folder of your wiki (usually rootdirectory/extensions)
*use ''git clone https://github.com/SrikarKashyap/FindRefs'' for cloning the directory
*open your LocalSettings.php page and update with the line 'wfLoadExtension( 'FindRefs' )';
*Navigate to 'Special:FindRefs' and provide your arguments for searching

#updation
*Use 'git pull origin master' for getting the updated version of the extension
